This server application is written in Ruby on Rails.
Ruby version for development: 2.0.0p481
Ruby version for production: 2.2.1p85
Rails version: 4.2.0
AngularJS version: 1.3.15

To run this application, you may need to have Rails correctly installed first.
Ruby on Rails: http://rubyonrails.org/download/
Then
1. run 'bundle install'
2. run 'bower install'
3. run 'rake db:migrate' or 'bundle exec rake db:migrate'
4. Done!
